List of steps for install
	1).Search "For Developers Settings"
		-Under "Use developer features" turn on sideload apps
	2).Install Python 2.7. Make sure you choose "Install for all users" 
	3).Run TrustbaseInstaller.exe as Adminstrator. 
	4).Run the windows shortcut "Run TrustBase". On the first run the shortcut will install the Trustbase Plugin Manager and all the python dependencies needed.
	5).Run "Run TrustBase" to start trustbase

Shortcuts in the start menu:
	-Run TrustBase (1st time): Installs the TrustBase Plugin Manager and python dependencies
	-Run TrustBase (After successful install): Starts the kernel module the policy engine in the background
	-Stop TrustBase: Stops the Policy engine and the kernel module
	-TrustBase Plugin Manager: Manages what certificate checks get run

Uninstalling:
	-Can be done by running the TrustbaseInstaller.exe as Adminstrator again or by going to add/remove programs
